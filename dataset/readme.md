# Dataset 
If you want to add a dataset, create a folder and add file `dataset.txt` and file `keywords_dict.txt` under the folder to record all data and keyword lists respectively.

for example, a file `dataset.txt` will be like this:
```bash
JazzClub 40.733596 -74.003139
Gym 40.758102 -73.975734
IndianRestaurant 40.732456 -74.003755
IndianRestaurant 42.345907 -71.087001

```
